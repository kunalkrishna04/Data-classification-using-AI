
# 📊 Data Classification Using AI

## 📌 Overview
Data Classification Using AI is a machine learning project that automatically classifies data into predefined categories based on input features. The system uses a classification algorithm to analyze the dataset, train a predictive model, and classify new data accurately.

## 🎯 Objectives
- Build an AI model for data classification.
- Train the model using labeled data.
- Predict the correct class for new input data.
- Evaluate the model's performance using standard metrics.

## ✨ Features
- Data preprocessing and cleaning
- Feature selection
- Model training and testing
- Automatic prediction
- Performance evaluation (Accuracy, Precision, Recall, F1-Score)
- Easy-to-use Python implementation

## 🛠️ Technologies Used
- Python
- Pydroid 3
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## ⚙️ Workflow
1. Load the dataset.
2. Clean and preprocess the data.
3. Split the dataset into training and testing sets.
4. Train the AI classification model.
5. Test the model using unseen data.
6. Evaluate the model's performance.
7. Predict the class of new data.

## 🤖 Algorithms Used
- Decision Tree
- Random Forest
- Logistic Regression
- Support Vector Machine (SVM)


## 📈 Results
- High classification accuracy on the test dataset.
- Efficient prediction of new data samples.
- Performance evaluated using confusion matrix and classification report.
